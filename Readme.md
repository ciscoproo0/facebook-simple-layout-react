#### Try it for yourself

clone the project and run using Yarn:

Download all dependencies: 

```bash
yarn 
```
Run the project
```bash
yarn dev 
```
or NPM

Download all dependencies
```bash
npm install
```
Run the project
```bash
npm run dev
```
